About Website Indexer

Website Indexer is a comprehensive directory of websites and applications for manga, light novels, movies, anime, learning resources, and useful apps.

Our mission is to provide a single destination where users can discover and access their favorite online resources easily.

Features:

  • Curated collection of websites
  • Real-time site health monitoring
  • Bookmarking functionality
  • Dark/Light theme support
  • Advanced search and filtering
  • NSFW content warnings
  • GitHub integration for community contributions

Version: 2.0
